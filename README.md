# Aussie Primary Academy Worksheet App V1

Basic free static worksheet generator framework.

## Included
- Grade selector
- Subject selector
- Topic selector
- ACARA v9.0 code field
- Worksheet seed/new version button
- A4 printable worksheet preview
- Print / Save as PDF using browser print
- Mandatory footer and copyright text
- Brand design:
  - Header/Footer: #CDB4DB
  - Accent: #FFD6A5
  - Border: #A2D2FF
  - Text: #333333
  - Font: Quicksand

## Files
- index.html
- styles.css
- worksheets.js
- app.js
- README.md

## How to use
Open `index.html` in Chrome. Select grade, subject and topic. Click Print / Save as PDF.

## How to deploy free
Upload all files to GitHub repository, then enable GitHub Pages.
